About The Project

Considering the high number of accidents caused by drowsy drivers leading to microsleep episodes, there is a need for measures to prevent or reduce such incidents. Therefore, this writing and project are expected to contribute to a reduction in traffic accidents caused by microsleep by developing a deep learning-based software that aims to provide alerts or alarms to detect microsleep using the Convolutional Neural Network (CNN) algorithm and the OpenCV library.

The goal of this undergraduate thesis project is to complete the bachelor's degree and create software capable of assisting in alerting drivers experiencing brief periods of sleep while driving (microsleep).
